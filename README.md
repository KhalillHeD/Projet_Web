Choix du Framework
Frontend : React + TypeScript + Vite

    React 18 : Bibliothèque moderne pour interfaces utilisateur, composants réutilisables

    TypeScript : Typage statique pour une meilleure maintenance et détection d'erreurs

    Vite : Build tool ultra-rapide avec rechargement à chaud

    Tailwind CSS : Framework CSS utilitaire pour un design cohérent et responsive

Backend : Django + Django REST Framework

    Django : Framework Python robuste avec ORM intégré

    Django REST Framework : Création d'APIs RESTful avec sérialisation

    Architecture MVT (Model-View-Template) éprouvée

    Sécurité intégrée : CSRF, authentification, validation

 Fonctionnalités Développées
    Frontend (Complété)

    Authentification : Page d'inscription et connexion

    Gestion des entreprises : Tableau de bord multi-business

    Système de facturation :

        Liste des factures avec recherche/filtrage

        Statuts (payée, impayée, en attente, retard)

        Création de nouvelles factures

        Téléchargement PDF

    Navigation avancée : Router personnalisé avec paramètres dynamiques

    UI/UX moderne :

        Design system cohérent

        Animations fluides

        Interface responsive

        Composants réutilisables

  Backend (À connecter)

    API RESTful : Endpoints pour toutes les fonctionnalités

    Modèles de données : Business, Factures, Clients

    Authentification : Sessions ou tokens JWT
